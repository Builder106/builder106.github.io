# Logo sources

Vendored (not hotlinked) so the page stays self-contained and doesn't depend on a third-party CDN staying up. Used for identification only — this project isn't affiliated with, endorsed by, or sponsored by any of these projects.

| File | Mark | Color | Source |
|------|------|-------|--------|
| `gltf-blender-io.svg` | glTF | `#87C540` | [Simple Icons](https://simpleicons.org/) (CC0); color cross-checked against the official mark in [`KhronosGroup/glTF`](https://github.com/KhronosGroup/glTF/blob/main/specification/figures/glTF_RGB_June16.svg) (`#86C540`) |
| `blender.svg` | Blender | `#E87D0D` | Simple Icons (CC0); color cross-checked against [blender.org's brand guidelines](https://www.blender.org/about/logo/) via independent citers |
| `cpython.svg` | Python | `#3776AB` | Simple Icons (CC0) — the standard single-accent Python blue; the official two-tone `python.org` logo uses `#306998`/`#FFD43B` |
| `neovim.svg` | Neovim | `#57A143` | Official mark from [`neovim.io/logos/neovim-logos.zip`](https://neovim.io/logos/neovim-logos.zip) (`neovim-mark-flat.svg`), by Jason Long, [CC BY 3.0](http://creativecommons.org/licenses/by/3.0/) |
| `pwndbg-wordmark.png` | pwndbg | `#8F2BF5` | Cropped from the `pwndbg/pwndbg` README's own banner image (a glitch-pixel "PWNDBG>" lockup); color sampled directly from the dominant pixel value in that crop. pwndbg has no dedicated logo file or brand guidelines page — this wordmark banner is the closest thing to an official mark, so it gets its own wide treatment in the account header instead of being forced into the square icon slot the other four use. |
| `pwndbg-favicon.png` | pwndbg | — | The `>` glyph from the same wordmark, extracted from [pwndbg.re](https://pwndbg.re/stable/assets/favicon.ico)'s own favicon and trimmed to its bounding box. Square, so it fills the compact trial-balance slot the wide wordmark doesn't fit. |

`pwndbg-wordmark.png` is a wide lockup (~4.8:1), not a compact square mark like the other four, so it appears only in the account header, where there's room for it; the trial balance uses `pwndbg-favicon.png` instead.

`gltf-badge.svg` is unused (a duplicate fetched during logo research) and can be deleted.
